Solution: Console-ation Prize
Answer: EVALUATION

Written by Lewis Chen

When opening the puzzle, we are greeted with three images and a larger canvas which has a single dot and line drawn on it. We can make a few observations, such that the dot appears at the endpoint of the broken line in the other three images, and that clicking on any image overlays that image on the canvas, but there’s no discernable way to interact with the puzzle.

Looking at the contents of the three images, they appear to draw out the characters F, 1, and 2. Combined with the title, this suggests pressing F12, which opens up the developer tools sidebar in most modern web browsers. In particular, we want to go to the console tab, where we are greeted with a prompt. As it turns out, we can invoke the functions a(), b(), c(), …, z() to change the drawing, with x() clearing the canvas as stated by the prompt.

Specifically, the behavior of the functions is as follows: The first, and every odd numbered invocation thereafter, function dictates the length of the segment. If this is not supplied, the segment continues until it goes off-screen. The second, and every even numbered invocation thereafter, function dictates the angle bearing for the next segment. If this is not supplied, the segment stops.

Since the three images appear to be generated using these functions, we can try to replicate the images using the functions. One way to do this is by trying functions until the length matches the first segment, then going to the next function and so forth. Another way is to measure lengths and angles in the image first and then try to match up the functions to these values.

It is not necessary to determine the lengths and angles for all functions, but a full table can be found in the appendix.

Either way, we can determine that the three images are separately generated via:

a() n() s() w() e() r()
i() s()
e() v() a() l() u() a() t() i() o() n()

The answer is EVALUATION.

Author’s Notes

The drawing mechanism was inspired by the fact that all three words of ANSWER IS EVALUATION are even length. Viewing the source is certainly possible, and there are deobfuscation tools available. This can help understand the behavior, though the answer isn’t clearly found in the source, as the images are just images. However, if you’re not careful, you might fall into a trap!
