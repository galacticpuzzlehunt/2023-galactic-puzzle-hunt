We see that 1) There are a very limited set of names among the faces represented (Abraham/Abe, Alexander/Alex, Andrew, Thomas/Tom, and George), and 2) Within a line, the names are always in the same order (e.g. when George appears, it’s always at the end of a line). This suggests the main aha, which is that each of the names is also the first name of a person depicted on a U.S. banknote: George Washington ($1), Thomas Jefferson ($2), Abraham Lincoln ($5), Alexander Hamilton ($10), and Andrew Jackson ($20).

We can now sum up the dollar values of the pictures in each line and convert the totals to letters with A1-Z26.
