Solution: Mastery Tree
Answer: CAPTAIN PI CARD

Written by Lewis Chen, Anderson Wang, and Violet Xiao

The meta uses the 18 answers placed on the mastery tree, as well as the mastery tree itself.

The answers are:

CIRCUIT
EJECTION
EVALUATION
INTEGRATED
LOTTERY
MESSAGE
PAYMENT METHOD
PERMANENT
PLASTIC
PLAYOFFS
QUALIFIER
REPETITION
RESIDENCY
SCHOLASTIC
SOCCER
SPACED
TICKET
VACATION

First, we notice that we can pair up some of the answers to make nice two-word phrases (LOTTERY TICKET, PERMANENT RESIDENCY, etc.), as well as some phrases that aren’t in the dictionary but make semantic sense together (PLAYOFFS QUALIFIER, PLASTIC PAYMENT METHOD, etc.). After finding enough pairs, we realize that each two-word phrase clues a kind of “card”:

Answer 1 Answer 2 Card
PLASTIC PAYMENT METHOD CREDIT card
SPACED REPETITION FLASH card
PERMANENT RESIDENCY GREEN card
VACATION MESSAGE POSTcard
SOCCER EJECTION RED card
SCHOLASTIC EVALUATION REPORT card
LOTTERY TICKET SCRATCH card
INTEGRATED CIRCUIT SMART card
PLAYOFFS QUALIFIER WILD card

(Note that pairing up answers is slightly motivated by the fact that there are exactly twice as many answers as masteries.)

Next, we notice that the words that precede “card” are synonyms of words in the mastery names (specifically the first word in each mastery name), which lets us match them up:

Card Mastery
Credit card Reputation Boost
Flash card Spark of Inspiration
Green card New Field
Post card Station Master
Red card Cardinal Director
Report card Rumor Has It…
Scratch card Erase Pain
Smart card Resourceful
Wild card Crazed Frenzy

Now that we have a pair of answers matching with each mastery, we can try to write the answers in the mastery tree grid so that each pair intersects at their corresponding teal square (for example, we want VACATION and MESSAGE to intersect at the “Station Master” square, because VACATION MESSAGE corresponds to a post card, and post is a synonym of station). This turns out to be doable in a unique way:

These answers are all connected in a “chain”, starting from EJECTION and ending at INTEGRATED. Reading the letters at the teal intersections from the beginning to the end of the chain gives CAPTAIN PI. We can add “card” to the end one last time to get the object we need to restore peace: the CAPTAIN PI CARD (a reference to Captain Picard from Star Trek).
